Voice Command System

The system is listening for the following keywords

Module You Say... System Action
NAV "open [images/videos/music/docs]" Switches to library view
NAV "open [camera/trash]" Launches system apps
CAM "start camera" Initializes video stream
CAM "take photo" Captures a still image
CAM "start/stop recording" Controls video capture
MEDIA "play video [name]" Searches & streams video
MEDIA "play song [name]" Initializes audio playback
CTRL "back" Goes to previous screen
CTRL "stop" Terminates all active tasks
